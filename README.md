# About InsightSearch
InsightSearch is built with Streamlit, and allows you to upload text files and perform powerful Boolean searches to find specific information within your documents. 
The app features a user-friendly interface and supports advanced search queries with AND, OR, and NOT operations.

## Key Features
- Easily upload multiple text files to create your searchable document repository.
- Perform complex searches using AND, OR, and NOT to quickly find relevant information.
- View the contents of matching documents directly within the app.
- Share your feedback to help improve the app's functionality.
